## nvm

先卸载掉已有的npm然后安装

mac遇到的问题 [zsh: command not found: nvm](https://blog.csdn.net/zm_miner/article/details/124850215)

```js
nvm current 当前版本号
nvm list 已安装的node
nvm list available 可安装的nvm
nvm i 版本号
nvm use 版本号

```
