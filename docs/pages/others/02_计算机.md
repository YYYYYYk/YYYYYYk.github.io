# 计算机

## mac
- 显示隐藏文件   `Command + Shift + .`

## dos常用命令
### 全局
``` js
  pwd  //显示当前路径
  cd ～  //进入根路径
  ls 展示当前路径下的文件。-a //包括隐藏文件
  clear //清屏
  touch ～/.zshrc  //在根路径创建.zshrc文件
  clear // 清屏
  who am i // 查看当前用户
  exit // 登出
```
### 文件相关
