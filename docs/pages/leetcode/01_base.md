## 复杂度
### 时间复杂度
**常见时间复杂度：**
O(1) 常量
O(log(n)) 换底公式,忽略对数的‘底’，统一记作O(log(n))
O(n) 
O(n²)

**递归的时间复杂度：**
如： 斐波那契数列。
O(2ⁿ)，指数级别的时间复杂度开销大
### 空间复杂度
常见空间复杂度：
O(1) 算法执行的临时空间不随n的大小而变化，如定义常量
O(n) 如申请长度为n的数组
对数阶复杂度非常少，而且空间复杂度相对时间复杂度分析简单的多，不在多阐述
