# CDN

content delivery network 内容分发网络

解决了：1. 物理距离问题 2. 用户运营商和业务服务器运营商 3. 业务服务器的带宽受限，处理能力能有限，难以处理海量用户请求

优势：1. 网站加速（静态） 2. 下载 3. 音视频加速 4. 全站加速 ECDN
